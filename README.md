この研究計画書ドリルでは、例として3つのテーマを挙げています。テーマをあなた自身が変えてもかまいません。
　初心者にとっては研究の足掛かりとして、このテーマの語り口はとても重要です。
　”何をすると（何が）”　”誰の”　”何に及ぼす影響”　（を調べた研究という意味）　として（実験の流れが題目だけで理解できるように設定すると、自分も他人も分かりやすいタイトルになります。

仮説は　これまで　論文では　はっきりしないこと、効果や影響が、正反対のものが書かれているようなものに対して、
きっと、条件が異なると、結果がされるのではないか、とか　対象の年齢が違うと、異なる効果があるのではないか、性別が異なると効果が変わるのではないか、というような発想を生かして、先行研究で結論がはっきりしないことを　自分の知識と、やはりいろいろと調べて、新しい考え方や、まだ行われていない条件で実施すると、”おそらく　こうなる”というものを設定します。

後は、先行研究でおこなれている対象、方法、統計手法を踏襲するか、それらのうち、どれかを変えて実験や観察、調査を行う手順を書いていきます。

人を相手にした場合、倫理審査を受ける必要性があります。の研究本当に学部の卒研で行えるのか？その人数で本当に統計学的有意差が得られるのか。被検者に無理な侵襲が加えられるなら、計画を変更しないといけないのではないか、PTの免許がいるのではないかなどが検討されます。
